argus

Argus - An easy logging module.

Argus makes logging easy and powerful. It provides a clean API for logging with both console and file output, automatic function call tracking, timing decorators, and more.

Key Features:
• Simple logging: info(), debug(), warning(), error(), critical()
• File and console output with different formats
• Automatic caller information (file, function, line number)
• Decorators for function call logging and timing
• Custom extra data support with clean structure
• JSON file output for easy parsing
• Human-readable console output
Quick Start:
>>> import argus
>>> argus.set_log_directory("logs")  # if you want to log to a file
>>> argus.max_logs(10)               # if you want to limit the number of logs
>>> argus.log_level(argus.DEBUG)     # set to level you want to see
>>> argus.info("Hello, world!")
>>> argus.info("User action", user_id=123, action="login")

With a couple of environment variables set, you can set up some defaults and make this even easier. Either through the regular environment, or .env files if you use dotenv. With those defaults set how you like (see below), you can then just do:

>>> import argus
>>> argus.info("Application starting")
Log Messages:

Logs can include both messages and extra data as named parameters. These additional parameterscan also be used with the strings. Both of these will result in the same logging message, for example:

>>> argus.info("User logged in: $user from $ip", user=user_id, ip=ip_address)
>>> argus.info(f"User logged in: {user_id} from {ip_address}", user=user_id, ip=ip_address)

This allows debug messages to simply be a string template and stored centrally, if useful. From a very slight performance perspective, using templates is better since the template is only evaluated if the log level would report the message, whereas the f-string is evaluated either way.

Defaults:

Defaults use environment variables, overriding with a .env file if it exists and dotenv is installed.

• Log level:
    Environment variable LOG_LEVEL or defaults to ERROR if not set

• Log directory:
    Environment variable LOG_DIRECTORY or defaults to not logging to file

• Max logs:
    Environment variable MAX_LOGS or defaults to keeping all logs

>>> argus.debug("Debug info")

>>> argus.error("Something went wrong")
>>> argus.critical("System failure!")

32
33    Returns:
34        None

37        >>> import argus
38        >>> argus.log(argus.DEBUG, "This is a debug message")

40        >>> argus.log(argus.WARNING, "This is a warning message", user_id=123, ip="192.168.1.1")
41    """
42    from .__init__ import logger  # pylint: disable=C0415
43    if level < logger.level:
44        return
45    caller_info = logger.findCaller(stack_info=False, stacklevel=3)
46    full_path = caller_info[0]
47    project_root = os.getcwd()
48    relative_path = os.path.relpath(full_path, start=project_root)

50    extra = {

52        "caller_func": caller_info[2],
53        "caller_lineno": caller_info[1],
54    }

56    # Add custom extra fields under a single known key
57    if extra_fields:
58        extra["extra_data"] = extra_fields
59        message = string.Template(message).safe_substitute(extra_fields)
60    logger._log(level, message, args=(), extra=extra)  # pylint: disable=W0212

180    """Decorator to log function calls with arguments and return values.
181
182    Args:
183        func: The function to log.
184
185    Returns:
186        The wrapped function.
187
188    Example:
189        >>> import argus
190        >>> @argus.log_function_call
191        >>> def my_function(a, b):
192        >>>     return a + b

196        ```
197        14:12:28 [DEBUG] main.py.my_function:200 - Calling function: my_function

199        14:12:28 [DEBUG] main.py.my_function:200 - Keyword arguments: {}

201        ```
202    """

204    def wrapper(*args, **kwargs):
205        debug(f"Calling function: {func.__name__}")
206        debug(f"Arguments: {args}")

211            return result
212        except Exception as e:

218def log_timing(func: Callable) -> Callable:
219    """Decorator to log function execution time.
220
221    Args:
222        func: The function to log.
223
224    Returns:
225        The wrapped function.
226
227    Example:
228        >>> import argus
229        >>> @argus.log_timing
230        >>> def my_function(a, b):
231        >>>     return a + b
232        >>> my_function(1, 2)

234        This will log something like the following:
235        ```
236        14:12:28 [DEBUG] main.py.my_function:200 - Function my_function took 0.00 seconds

239        Note: you can combine this with ```log_function_call``` to get a

248    def wrapper(*args, **kwargs):
249        start_time = time.perf_counter()

252        elapsed_time = end_time - start_time
253        info(f"Function {func.__name__} took {elapsed_time:.2f} seconds")
254        return result
